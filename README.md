# final-project-hafen340

Final project for CSE 340: a **book discovery and content-rating platform** that helps readers understand what they are getting into before picking up a book.

## Project Summary

Users can:
- Browse books in the catalog
- View community-submitted content ratings, including:
  - Spice level
  - Violence
  - Language
- Leave book reviews
- Request books that are not yet in the catalog

This project is built for readers who want more content transparency than a typical star rating provides.
